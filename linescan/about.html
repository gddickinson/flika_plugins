Linescan

Interactive line profile tool for extracting intensity values along a drawn line ROI. Displays the intensity profile in a real-time plot window that updates as you move the line or change frames.

Features

  • Draw a line ROI on any image window
  • Real-time intensity profile plot
  • Updates live as you move the ROI or navigate frames
  • Configurable plot display options

Usage

  1. Open an image in flika
  2. Go to Plugins > Linescan
  3. Draw a line on the image
  4. The intensity profile appears in a separate plot window
  5. Move the line or scroll frames to update the profile
