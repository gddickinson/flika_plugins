Detect Puffs (GD Edit)

Comprehensive calcium puff detection and analysis pipeline. Detects, characterizes, and quantifies calcium release events (puffs) in fluorescence microscopy data.

Features

  • Detection pipeline — threshold supra-threshold pixels in blurred movies
  • Clustering — connected component analysis to group events
  • Gaussian fitting — 2D Gaussian fit for sub-pixel localization
  • Spatial grouping — group events into release sites
  • Per-site statistics — amplitude, duration, frequency analysis
  • Manual selection — interactive event selection and validation
  • Puff simulator — generate synthetic test data

Usage

  1. Open a calcium imaging time series in flika
  2. Go to Plugins > Detect Puffs
  3. Configure detection parameters (threshold, blur sigma)
  4. Run detection to identify events
  5. Review and classify detected puffs
  6. Export results and statistics
