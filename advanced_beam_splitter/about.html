Advanced Beam Splitter

Comprehensive dual-channel TIRF microscopy image alignment and processing plugin. Designed for experiments using a beam splitter to simultaneously image two channels (e.g., red/green fluorescence) on the same camera sensor.

Features

  • Geometric transformations — XY translation, rotation, and scaling to align channels
  • Background subtraction — rolling ball, Gaussian, and manual ROI methods
  • Photobleaching correction — exponential fit and histogram matching
  • Intensity normalization — match channel intensities for overlay
  • Auto-alignment — phase cross-correlation for automatic registration
  • Live preview — real-time red/green overlay with keyboard controls

Usage

  1. Open a dual-channel image (split left/right or top/bottom)
  2. Go to Plugins > Advanced Beam Splitter
  3. Adjust alignment using arrow keys or auto-align
  4. Apply background subtraction and bleaching correction
  5. Export aligned channels
