Timestamp

Adds a dynamic elapsed-time label (in milliseconds) to any flika image window. The timestamp updates automatically as you scroll through frames, using the window's frame rate and current frame index to compute elapsed time.

Features

  • Adds a text overlay showing elapsed time in milliseconds
  • Automatically updates as you navigate through frames
  • Uses the window's configured frame rate for accurate timing
  • Connects to the window's sigTimeChanged signal for live updates

Usage

  1. Open a 3D image stack (time series) in flika
  2. Go to Plugins > Timestamp
  3. The timestamp overlay will appear on the current window
  4. Scroll through frames to see the timestamp update

Notes

The frame rate is read from the window's settings. Make sure the frame rate is correctly set (via File > Settings) for accurate time display.
