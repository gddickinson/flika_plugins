Localizations & Tracks Plotter

Comprehensive visualization and analysis tool for single-molecule localization and tracking data. Loads data from ThunderSTORM or flika Pynsight and provides interactive scatter plots, track overlays, heat maps, diffusion analysis, and classification.

Features

  • Data loading — ThunderSTORM CSV, flika CSV, JSON formats
  • Scatter plots — interactive point localization display with filtering
  • Track overlay — visualize particle trajectories on images
  • Heat maps — density and intensity heat map generation
  • Diffusion analysis — MSD, SLD, CDF analysis per track
  • Classification — SVM-based track classification
  • Flower plots — origin-centered track display
  • Color maps — multiple scientific color map options
  • Filtering — filter by intensity, sigma, track length, etc.
  • Export — save filtered data to CSV

Usage

  1. Go to Plugins > Locs and Tracks Plotter
  2. Load a CSV file with localization or track data
  3. Use the dockable panels to filter, visualize, and analyze
