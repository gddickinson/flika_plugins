QUICK INSTALLATION GUIDE
========================

PIEZO1 Cluster Analyzer Plugin for FLIKA
Pathak Lab, UC Irvine

INSTALLATION STEPS:
-------------------

1. Find your FLIKA plugins directory:
   
   Windows: C:\Users\YourName\.FLIKA\plugins\
   Mac:     ~/.FLIKA/plugins/
   Linux:   ~/.FLIKA/plugins/

2. Copy the entire "piezo1_cluster_analyzer" folder to the plugins directory

3. Restart FLIKA

4. The plugin will appear under: Plugins → PIEZO1 Cluster Analyzer

REQUIRED FILES (all included in this folder):
----------------------------------------------
✓ piezo1_cluster_analyzer.py (main plugin code)
✓ __init__.py (Python package file)
✓ info.xml (plugin metadata)
✓ about.html (plugin description)
✓ README.md (full documentation)

QUICK START:
-----------
1. Open a PIEZO1 fluorescence image in FLIKA
2. Go to Plugins → PIEZO1 Cluster Analyzer
3. Click "Run Analysis" with default settings
4. Click "Visualize Results" to see the analysis

For detailed usage instructions, see README.md

TESTING THE PLUGIN:
------------------
After installation, test with a sample PIEZO1 image:
1. Load any image with bright puncta/spots
2. Open the plugin GUI
3. Try running with default parameters
4. Check that visualization window opens

TROUBLESHOOTING:
---------------
If the plugin doesn't appear:
- Make sure all 5 files are in the same folder
- Check the folder name is exactly: piezo1_cluster_analyzer
- Verify the plugin directory path is correct
- Restart FLIKA completely

If you get import errors:
Install missing packages:
pip install numpy scipy scikit-learn scikit-image pandas matplotlib --break-system-packages
