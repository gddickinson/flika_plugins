Overview

This FLIKA plugin provides comprehensive tools for analyzing the relationship between cell edge movement and PIEZO1 mechanosensitive channel protein intensity using advanced image processing techniques.

ðŸ" Analysis Method: Vertical Displacement X-Axis Full

The plugin implements the vertical_displacement_xaxis_full method, which:

  • Processes every x-pixel position across the cell
  • Finds the uppermost (minimum y) edge point at each x-coordinate
  • Calculates displacement between consecutive frames
  • Creates comprehensive 2D movement maps
  • Correlates movement with PIEZO1 intensity at sampling regions

Workflow

  1. Setup: Select PIEZO1 signal and mask windows from FLIKA
  2. Edit Masks (Optional): Use drawing tools to refine cell masks
  3. Configure: Set analysis parameters and temporal direction
  4. Run Analysis: Execute analysis with progress monitoring
  5. Visualize Results: Navigate frames and view correlations
  6. Export: Save results as CSV and summary plots

Temporal Directions

Future Direction: Intensity at frame N predicts movement from N to N+1

Use when: Testing if current protein localization influences subsequent cell movement

Past Direction: Movement from N-1 to N correlates with intensity at N

Use when: Testing if recent cell movement affects current protein localization
