Overlay

Creates RGB channel overlays from up to three separate flika windows (red, green, blue). Includes gamma correction and frame linking for synchronized multi-channel visualization.

Features

  • Combine up to 3 windows into an RGB composite
  • Per-channel gamma correction
  • Linked frame navigation across channels
  • Live preview as you adjust settings

Usage

  1. Open 2-3 single-channel images in flika
  2. Go to Plugins > Overlay
  3. Assign each window to R, G, or B channel
  4. Adjust gamma and intensity as needed
  5. The overlay window updates in real time
