Kymograph Analyzer

Generates kymographs (space-time intensity maps) from line ROIs drawn on 3D time-series image stacks. Features automatic event detection, velocity measurement, statistical analysis, and comprehensive visualization.

Features

  • Kymograph generation — from line ROIs with configurable width
  • Event detection — peak, threshold, and z-score methods
  • Velocity measurement — gradient-based velocity estimation
  • Statistical analysis — event amplitude, duration, frequency
  • Visualization — matplotlib-based plots with export
  • Export — CSV and NPY output formats

Usage

  1. Open a 3D image stack in flika
  2. Draw a line ROI across the region of interest
  3. Go to Plugins > Kymograph Analyzer
  4. The kymograph is generated along the ROI
  5. Use the analysis tools to detect events and measure velocities

Ideal for Ca2+ flickers, membrane protein dynamics, vesicle trafficking, and wave propagation studies.
