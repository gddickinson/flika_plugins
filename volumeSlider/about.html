Volume Slider

A comprehensive 4D volume viewer for light-sheet and volumetric microscopy data. Reshapes 3D stacks (frames × height × width) into 4D volumes (volumes × slices × height × width) and provides interactive slice-by-slice navigation.

Features

  • 4D volume reshaping — converts 3D stacks to 4D volumes with configurable slices-per-volume
  • Slice slider — navigate through Z slices within each volume
  • ΔF/F0 ratio — compute fluorescence change relative to baseline
  • Baseline subtraction — configurable baseline window
  • Data type conversion — convert between float16, float32, float64
  • 3D OpenGL visualization — interactive 3D volume rendering
  • Max/mean projection — project volumes along the Z axis
  • Batch processing — process multiple files with the same settings
  • Overlay support — overlay multiple volume channels
  • Export — save to numpy arrays or new flika windows

Usage

  1. Open a 3D image stack in flika
  2. Go to Plugins > Volume Slider
  3. Set the number of Z slices per volume
  4. Use the slice slider to navigate through the volume
  5. Apply ΔF/F0 or other processing as needed

Requirements

  • PyOpenGL (optional, for 3D viewer)
