üîß Installation

Method 1: FLIKA Plugin Manager (Recommended)

  1. Open FLIKA
  2. Go to Plugins ‚Üí Plugin Manager
  3. Search for "Volume Slider Professional"
  4. Click Install
  5. Restart FLIKA
  6. Find the plugin under Plugins ‚Üí Volume Slider Professional

Method 2: Manual Installation

  1. Download the plugin package from GitHub
  2. Extract to your FLIKA plugins directory: ~/.FLIKA/plugins/
  3. Restart FLIKA

System Requirements

  • FLIKA: Version 0.2.23 or higher
  • Python: 3.7+ (3.8+ recommended)
  • Memory: 4GB RAM minimum, 8GB+ for large datasets
  • Graphics: OpenGL 3.3+ compatible card recommended
  • Storage: 1GB available disk space

üöÄ Quick Start

Basic Usage

  1. Load your lightsheet data in FLIKA
  2. Launch Volume Slider Professional from the Plugins menu
  3. The plugin will automatically load your data
  4. Use the dockable panels to adjust visualization and processing settings
  5. Navigate through volumes using the Volume Control panel
  6. Apply processing operations from the Processing panel
  7. Analyze data using the Analysis panel

Generate Test Data

  1. Launch the plugin (data loading is optional)
  2. Go to Data ‚Üí Generate Test Data...
  3. Choose pattern type (Ca¬≤‚Å∫ puffs, waves, cell migration, etc.)
  4. Adjust parameters and click Generate
  5. Explore the generated data with different visualization modes

Memory Usage: Large datasets (>2GB) may require significant RAM. The plugin includes memory management features, but ensure adequate system memory for optimal performance.
Graphics Requirements: 3D visualization requires OpenGL 3.3+. On older systems, some 3D features may be limited or unavailable.

üîç Supported File Formats

Input Formats

  • TIFF/TIF: Standard microscopy format, multi-page stacks
  • NumPy (.npy): Python array format for processed data
  • HDF5 (.h5, .hdf5): Large dataset format
  • FLIKA Windows: Direct import from FLIKA workspace

Output Formats

  • TIFF/TIF: Processed image stacks
  • NumPy (.npy): Numerical data arrays
  • Imaris (.ims): 3D visualization format
  • CSV/Excel: Analysis results and statistics
  • HDF5: Large processed datasets
