Annotator

Burns frame-number text annotations into each frame of a 3D image stack. Useful for creating labeled image sequences for presentations or publications.

Features

  • Renders frame index (0, 1, 2, ...) as text on each frame
  • Uses PIL/Pillow for high-quality text rendering
  • Creates a new annotated window (non-destructive)

Usage

  1. Open a 3D image stack in flika
  2. Go to Plugins > Annotator
  3. The plugin creates a new window with frame numbers burned in

Requirements

  • Pillow (PIL) for text rendering
