Video Exporter

Exports flika image stacks as MP4 videos with optional ROI zoom views, timestamps, and scale bars. Includes a dedicated ROI Video Exporter panel for detailed region-of-interest visualization and export.

Features

  • MP4 video export from any flika image stack via ffmpeg
  • ROI zoom view — magnified view of a selected region with independent controls
  • Timestamp overlay — configurable elapsed-time display
  • Scale bar overlay — calibrated scale bar with customizable size and position
  • Frame range control — export a subset of frames
  • Screen recorder — standalone utility for screen capture

Usage

  1. Open an image stack in flika
  2. Go to Plugins > Video Exporter
  3. Draw an ROI on the image to define the zoom region (optional)
  4. Configure timestamp, scale bar, and frame range as needed
  5. Click Export to save the video

Requirements

  • ffmpeg must be installed and accessible on the system PATH
  • opencv-python (optional, for screen recorder)
