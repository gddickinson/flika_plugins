Overlay Multiple Recordings

Overlays localization data from multiple STORM/PALM recordings onto a single image. Provides a dockable control panel for loading, transforming, filtering, and visualizing multi-recording datasets with heat maps and scatter plots.

Features

  • Load and overlay data from multiple CSV files
  • Coordinate transformation and alignment
  • Interactive scatter plot and heat map display
  • Filter by intensity, sigma, track length
  • Multiple color map options
  • Export combined datasets

Usage

  1. Go to Plugins > Overlay Multiple Recordings
  2. Load CSV files containing localization data
  3. Adjust transformations to align recordings
  4. Use the visualization panel to explore the combined data
