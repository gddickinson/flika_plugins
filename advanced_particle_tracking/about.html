üß¨ Scientific Foundation

1. Sub-Pixel Particle Detection

Implements hybrid approach combining:

  • Deep Learning Component: CNN-based density map regression for robust detection
  • Classical Component: Direct Gaussian fitting for optimal sub-pixel accuracy
  • Temporal Integration: Multi-frame information for improved detection
Based on: "Deep learning method for probabilistic particle detection and tracking in fluorescence microscopy images" and classical Gaussian fitting methods proven optimal for point sources.

2. Probabilistic Particle Tracking

Advanced tracking system featuring:

  • Bayesian Framework: Uncertainty quantification for each track
  • LSTM-style Memory: Temporal dependency modeling
  • Linear Assignment: Optimal particle-track associations
  • Predictive Modeling: Velocity and acceleration-based position prediction
Inspired by: "Deep probabilistic tracking of particles in fluorescence microscopy images" and Bayesian filtering approaches.

3. Machine Learning Classification

Comprehensive movement analysis using:

  • 17 Diffusional Features: Complete characterization including MSD, velocity, directionality
  • Unsupervised Clustering: Automatic identification of movement types
  • Confidence Scoring: Reliability assessment for each classification
  • Anomalous Diffusion Detection: Subdiffusive, superdiffusive, and confined motion
Based on: "Single-particle diffusional fingerprinting" and "Classification of diffusion modes in single-particle tracking data" methodologies.

üîß Usage Instructions

Access: Plugins ‚Üí Advanced Particle Tracking ‚Üí Track & Classify Particles

üìã Workflow:

  1. Load Data: Open time-series fluorescence microscopy data in FLIKA
  2. Configure Detection: Set sigma, threshold, and minimum particle size
  3. Set Tracking Parameters: Define max displacement and memory settings
  4. Enable Classification: Choose analysis window for feature extraction
  5. Run Analysis: Execute complete pipeline
  6. Review Results: Examine tracks, classifications, and confidence scores

‚öôÔ∏è Parameters

Detection Parameters:

  • Detection Sigma: Gaussian filter parameter (0.5-5.0, default: 1.5)
  • Intensity Threshold: Minimum particle intensity (0.01-1.0, default: 0.1)
  • Min Particle Size: Minimum particle diameter in pixels (1-20, default: 3)

Tracking Parameters:

  • Max Displacement: Maximum particle movement between frames (1-50, default: 10)
  • Memory: Frames to remember lost particles (0-10, default: 3)
  • Min Track Length: Minimum track length for analysis (3-100, default: 10)

Classification Parameters:

  • Enable ML Classification: Activate movement type analysis (default: true)
  • Analysis Window: Window size for feature extraction (5-50, default: 20)
