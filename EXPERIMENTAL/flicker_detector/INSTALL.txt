QUICK INSTALLATION GUIDE
========================

Flicker/Transient Detector Plugin for FLIKA
Pathak Lab, UC Irvine

INSTALLATION STEPS:
-------------------

1. Find your FLIKA plugins directory:
   
   Windows: C:\Users\YourName\.FLIKA\plugins\
   Mac:     ~/.FLIKA/plugins/
   Linux:   ~/.FLIKA/plugins/

2. Copy the entire "flicker_detector" folder to the plugins directory

3. Restart FLIKA

4. The plugin will appear under: Plugins → Flicker/Transient Detector

REQUIRED FILES (all included in this folder):
----------------------------------------------
✓ flicker_transient_detector.py (main plugin code)
✓ __init__.py (Python package file)
✓ info.xml (plugin metadata)
✓ about.html (plugin description)
✓ README.md (full documentation)

QUICK START:
-----------
1. Load a calcium imaging time series in FLIKA
2. (Optional) Draw ROIs over regions of interest
3. Go to Plugins → Flicker/Transient Detector
4. Set framerate (Hz)
5. Click "Detect Events" with default settings
6. Click "Visualize Results"

RECOMMENDED STARTING PARAMETERS:
-------------------------------
- Detection Method: threshold
- Threshold: 2.0 (for ΔF/F₀)
- Min Duration: 3 frames
- Max Duration: 100 frames
- Smoothing: 1.0
- Baseline: percentile (10%)
- Framerate: Your acquisition rate in Hz

TESTING THE PLUGIN:
------------------
After installation:
1. Load any time series with transient signals
2. Open plugin GUI
3. Enter your framerate
4. Click "Detect Events"
5. Check visualization window

TROUBLESHOOTING:
---------------

Plugin doesn't appear in menu:
- Check folder name is exactly: flicker_detector
- Verify all files are present
- Restart FLIKA completely
- Check FLIKA console for errors

No events detected:
- Threshold may be too high - try lowering to 1.5
- Check signal quality
- Try 'zscore' method with threshold 2.0-3.0

Too many false events:
- Threshold too low - increase to 3.0+
- Increase min_duration
- Apply smoothing (sigma 1.0-2.0)

Import errors:
pip install numpy scipy pandas matplotlib --break-system-packages

WHAT THIS PLUGIN DOES:
----------------------

Automatically detects and characterizes transient intensity increases 
(flickers, sparks, transients) in fluorescence time series.

Perfect for:
✓ Ca²⁺ flickers in PIEZO1 studies
✓ Localized calcium signals
✓ Mechanotransduction events
✓ Any rapid transient increases

Provides:
✓ Automatic event detection (4 methods)
✓ Comprehensive characterization (amplitude, duration, kinetics)
✓ Statistical analysis
✓ Raster plots
✓ Data export
