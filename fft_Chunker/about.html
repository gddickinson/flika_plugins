FFT Chunker

Loads CSV time-series data, splits traces into chunks, performs FFT on each chunk, averages power spectra across user-defined groups (baseline and up to 4 experimental conditions), and exports log10-transformed results for frequency analysis.

Features

  • Load time-series data from CSV files
  • Split traces into configurable chunk sizes
  • Compute FFT power spectra per chunk
  • Average spectra across baseline and experimental groups
  • Export log10-transformed power spectra

Usage

  1. Go to Plugins > FFT Chunker
  2. Load a CSV file with time-series data
  3. Set chunk size and group boundaries
  4. Run the FFT analysis
  5. Export results to CSV
