The Global Analysis Plugin

Uses the scipy.poly function to fit ROI traces to a polynomial function, and predict the fall and rise times of active sites.

To use the plugin:

  • Draw and plot ROI(s)
  • Select the ROI in the Trace Dropbox of the Global Analysis GUI
  • Check the "Show Trace ROI" Checkbox to enable the region selection on the trace window
  • Select the frame region that the activity occurs in. The table below will display information about the polynomial fit
  • Use the Measure Tool to enable distance logging and slope checking
  • The Log and Save Log buttons at the bottom function the same as in the measure tool, compiling a file of region statistics
