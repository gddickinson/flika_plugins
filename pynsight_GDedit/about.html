Pynsight (GD Edit)

A Python clone of Insight2 for analyzing 2D STORM super-resolution microscopy movies. Provides a complete pipeline from detection through linking and analysis, including Gaussian refinement, track linking, SLD histograms, MSD analysis, and CDF fitting.

Features

  • Detection — identify bright spots in STORM frames
  • Gaussian refinement — sub-pixel localization via 2D Gaussian fitting
  • Track linking — connect localizations across frames into tracks
  • SLD analysis — step length distribution histograms
  • MSD analysis — mean squared displacement for diffusion characterization
  • CDF fitting — cumulative distribution function analysis
  • Visualization — reconstructed images, track overlays, analysis plots

Usage

  1. Open a STORM movie in flika
  2. Go to Plugins > Pynsight
  3. Run detection to find spots
  4. Refine with Gaussian fitting
  5. Link into tracks and analyze
