Frame Remover

Removes periodic blocks of frames from image stacks. Given a start frame, end frame, block length, and interval, it deletes frames at regular intervals throughout the specified range. Useful for removing stimulus frames or other periodic artifacts.

Parameters

  • Start frame — first frame to begin removing
  • End frame — last frame in the removal range
  • Block length — number of consecutive frames to remove per interval
  • Interval — spacing between removal blocks

Usage

  1. Open a 3D image stack in flika
  2. Go to Plugins > Frame Remover
  3. Set the start, end, block length, and interval
  4. Click OK to generate a new stack with frames removed
