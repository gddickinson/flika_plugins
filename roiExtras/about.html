ROI Extras

Live ROI histogram and statistics tool. Displays a real-time histogram of pixel values within the current ROI, with autoscaling and ROI masking support.

Features

  • Live histogram of pixel values inside the selected ROI
  • Auto-scaling X axis option
  • Updates in real time as ROI is moved or resized
  • ROI-masked image preview

Usage

  1. Open an image in flika and draw an ROI
  2. Go to Plugins > ROI Extras
  3. The histogram window shows the pixel distribution within the ROI
  4. Move or resize the ROI to update the histogram
