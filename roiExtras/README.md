# ROI Extras

Live ROI histogram and statistics tool. Displays a real-time histogram of pixel values within the current ROI.

## Features

- Live histogram of pixel values inside the selected ROI
- Auto-scaling X axis option
- Updates in real time as ROI is moved or resized
- ROI-masked image preview

## Requirements

- [flika](https://github.com/flika-org/flika) >= 0.2.23

## Usage

1. Open an image in flika and draw an ROI
2. Go to **Plugins > ROI Extras**
3. The histogram window shows pixel value distribution within the ROI
4. Move or resize the ROI to update the histogram in real time
