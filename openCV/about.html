OpenCV Camera

Live webcam capture and AVI file import using OpenCV. Stream camera frames directly into flika windows for real-time processing and analysis.

Features

  • Live camera capture — stream webcam frames into flika
  • AVI file import — load video files via OpenCV
  • Basic filters — Canny edge detection, background subtraction

Requirements

  • opencv-python (pip install opencv-python)

Usage

  1. Go to Plugins > OpenCV
  2. Choose Start Camera to begin live capture
  3. Or choose Open File to load an AVI video
